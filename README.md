# Biology of Aging — Exam 1 Study Site

An interactive, self-contained study website for **BIO 41000 — Cell & Molecular
Biology of Aging** (Exam 1, Meetings 1–7): theories of aging, genomic instability
& DNA repair, telomeres & senescence, epigenetics, proteostasis, mitochondria &
oxidative stress, and nutrient sensing.

It runs entirely from static files — open `index.html` in any browser (desktop or
mobile). All content lives in `course-data.js`; lecture slide images are under
`slides/`. There are no external dependencies.

## Files
- `index.html` — the study-site engine (Learn / quiz / flashcards UI)
- `course-data.js` — all lecture content, quizzes, and flashcards
- `slides/L1`–`slides/L7` — lecture slide images

## A note on the materials

The slide images are course materials, included here **with the instructor's
permission for personal study use**. Please respect that scope and do not
redistribute them for other purposes.
